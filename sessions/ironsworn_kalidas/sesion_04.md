### Escena 9 *Los Espinos de Sangre*

```
@ Entrar en liza
d: Acción=Mente+2=5 [Sorprendido] vs Desafío=1,6 -> Éxito parcial
=> Nos preparamos para actuar tomando la iniciativa
```

```
@ Golpear
d: Acción=Hierro+1=3 [Maestría con el Hacha] vs Desafío=1,9 -> Éxito parcial
=>
  Impulso -1 (+7)
  Inflijo 2+1=3 Daño
  Pierdo la iniciativa
  [Track:Espinos de sangre|Peligroso|6/10]
```

Golpeé con fuerza mi hacha contra aquella criatura y su sangre roja me llenó la cara. Fui demasiado descuidado y mi posición ahora estaba expuesta.

```
@ Contratacar
d: Acción=Hierro+2=4 vs Desafío=7,10 -> Fallo
=> Salud -2 (2)

@ Sufrir daño
d: Acción=Salud+1=3 vs Desafío=3,9 -> Fallo
=> Impulso -1 (+6)
```

Aquella cosa me agarró el brazo y clavó sus espinas sobre mi piel. Me retorzí de dolor y giré hacia un costazo para intentar poder librarme. 

Decidí que lo mejor sería averiguar el punto débil de la criatura. Sabía que tenía una conexión especial con el misticismo, así que decidí detectar las fuerzas místicas.

```
@ Afrontar el peligro
d: Acción=Mente+1+3=7 [Avistado] vs Desafío=3,10 -> Éxito parcial
gen: Acción/Tema/Foco -> Restaurar/Estrategia/Restos
=>
  +1 Impulso (7)
  La base parece salir de un muro derribado
```

Aquella planta parecía surgir de detrás de un viejo muro derribado. Debía llegar allí y atacar desde la parte posterior. Pero no parecía tarea sencilla. Decidí resistir para poder esquivar su golpe y alcanzar la retaguardia.

```
@ Contratacar
d: Acción=Hierro+4=6 vs Desafío=7,10 -> Fallo
gen: Pagar el precio -> Un nuevo peligro es revelado
gen: Acción/Tema -> Manipular/Peligro
=> 
  Descubre mis intenciones
  -2 Impulso (5)
```

La planta hizo un movimiento defensivo y cortó mi camino hacia la parte de atrás. Con la ventaja perdida, lanzó un ataque.

```
@ Contratacar
d: Acción=Hierro+4=6 vs Desafío=3,5 -> Éxito total
=> 
  Inflijo 2 Daño
  Impulso +1 (6)
  Gano la iniciativa
  [Track:Espinos de sangre|Peligroso|10/10]

@ Terminar la lucha
d: Track=10 vs Desafío=4,7
=> Derrotado
```

Lancé un ataque directo contra el suyo, a modo desesperado y resultó en un golpe mortal. La planta cedió ante mi filo.

```
@ Curarme
d: Acción=Hierro+3=5 vs Desafío=1,9 -> Éxito parcial
=>
  Saco unas vendas y curo mis heridas
  +2 Salud (4)
  -1 Suministros (2)
```

Decidimos parar un poco para poder curar nuestras heridas. Usé algunos suministros que llevaba encima.

